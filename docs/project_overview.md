# U.S. Visa Salary Insights

This project provides a data-driven exploration of wage patterns and employment trends across various U.S. visa classifications. Using publicly available labor certification data, it highlights how visa type correlates with salary levels, job roles, employer demand, and geographic distribution.

## Purpose

The goal is to support researchers, policymakers, and job seekers by offering clear, accessible visualizations that reveal how immigration status intersects with labor market dynamics. All visualizations are generated using Python and standard data visualization libraries, with a focus on reproducibility and beginner-friendly documentation.

## Contents

The repository includes:
- Source code for data processing and visualization  
- An interactive HTML report (`salary_report_v1.html`)  
- PNG exports of key charts for quick reference  
- Documentation to guide exploration and extension
